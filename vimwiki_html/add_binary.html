两个二进制数相加

题目要求 ： 

Given two binary strings, return their sum (also a binary string).

For example,
a = "11"
b = "1"
Return "100".

注意的点 ：

  • 两个二进制数的长度不好把握 
class Solution {
public:
    string addBinary(string a, string b) {
        int m = a.length() - 1;
        int n = b.length() - 1;
        int c = 0;
        string output = "";
        while(m >= 0 || n >= 0 || c > 0)
        {
            c += (m >= 0 ? (a[m--] - '0') : 0);
            c += (n >= 0 ? (b[n--] - '0') : 0);
            output = char(c % 2 + '0') + output;
            c = c/2;
        }
        return output;
    }
};
